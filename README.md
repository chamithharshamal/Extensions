# Chrome Extensions Collection

A curated collection of modern, premium Chrome extensions built for productivity and seamless user experience.

## Extensions

### 1. [URL to Mobile QR](file:///f:/Projects/Extensions/url-to-mobile-qr)
Instantly generate a QR code for the current tab's URL to open it on your mobile device.
- **Features**: Instant QR, Clipboard Copy, Premium UI.
- **Tech**: JavaScript, HTML5 Canvas, CSS Grid.

---

## How to Install Any Extension
1. Open **Google Chrome**.
2. Navigate to `chrome://extensions/`.
3. Enable **Developer mode** (top right).
4. Click **Load unpacked**.
5. Select the folder of the extension you want to install (e.g., `url-to-mobile-qr`).
